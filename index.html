Asirvad Gold Loan Module

Worked in Gold Loan Management System for Asirvad Micro Finance, enabling customers to pledge gold as collateral and manage loans efficiently. The system streamlines loan disbursement, repayment tracking, interest calculations, and customer data management. The frontend was built using Angular to provide a responsive and intuitive user experience, while the backend utilized ASP.NET for secure and scalable operations. Oracle PL/SQL was used for efficient database management, ensuring smooth data handling and reporting. This module improved the loan processing workflow, enhanced operational efficiency, and elevated customer service quality.

Asirvad Sticker Module

Worked in Sticker Module for Asirvad Micro Finance to audit pledged gold and maintain real-time updates on its status. This module ensures accurate tracking of pledged assets, verifying whether they are available, under processing, or released. The frontend was implemented using Angular for a user-friendly and dynamic interface, while the backend was powered by ASP.NET to ensure secure and efficient operations. Oracle PL/SQL was employed for robust database management, supporting seamless data updates and reporting. This module improved the transparency and reliability of the gold audit process, enhancing both operational control and customer trust.

Asirvad Employee Self-Service (ESS) App

The app integrates multiple HRMS modules, such as attendance management and performance tracking, providing a unified platform for employees and managers.
